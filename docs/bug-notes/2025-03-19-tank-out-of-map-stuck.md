# Танк выезжает за границу карты и застревает

**Дата:** 2025-03-19  
**Статус:** исправлено

## Симптом

Корпус частично оказывается над/за пределами сетки (например у верхней границы), дальше не отъехать.

## Причина

Для выхода за карту `tankBrickCollisionIndex` возвращает `-2`. В `separateTankFromBricks` при `-2` центр танка ограничивали как **`Math.max(hw, …)` с локальными `hw = w/2`, `hh = h/2`** — это верно только для **нулевого** поворота. У повёрнутого OBB углы вылезают за пределы этого «квадрата»; кламп не возвращал центр в допустимую область, цикл мог крутиться без эффекта или оставлять танк частично снаружи.

Дополнительно при **повороте на месте** (A/D) центр не меняется, но OBB разворачивается — углы могут выйти за `[0, mapW]×[0, mapH]` без шага движения, если центр был у края.

## Фикс

- `tankObbWorldAabbHalfExtents(angle)` — полуоси **мирового** AABB вокруг повёрнутого корпуса (та же формула, что раньше для «раздутого» хитбокса, но только для границы карты).
- `clampTankCenterToMap()` — кламп `tank.x` / `tank.y` по этим полуосям.
- Вызов после **A/D**, после блока **enemy + separateTankFromBricks**; ветка `-2` в `separateTankFromBricks` вызывает `clampTankCenterToMap()`.

Файл: `client/src/game/gameClient.js`.

## Проверка

Подъехать к верхней границе, покрутить корпус, попытаться выехать наружу — центр остаётся так, чтобы весь OBB был внутри карты.
